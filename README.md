# Real-estate-crm 🏠🤖

**RECRM** is a high-performance, AI-driven Real Estate ERP/CRM architected to streamline the lifecycle of Philippine real estate leads. Built with **Next.js 15**, it bridges the gap between social media lead generation and financial closing through automated intelligence and secure, role-based operations.

---

## 🚀 Core Features

### 🧠 AI-Powered Lead Intelligence
- **Dynamic Temperature Scoring:** Automatically classifies leads as 🔥 **Hot**, ☀️ **Warm**, or ❄️ **Cold** based on engagement recency, budget, and interaction frequency via **Gemini 1.5 Flash**.
- **The Closer’s Corner:** A dedicated agent dashboard prioritizing the "Top 5" next-best-actions generated by AI.

### 🔒 Enterprise-Grade Security
- **PII Masking:** Personally Identifiable Information (Names, Phone Numbers) is masked for all users except the lead owner. 
- **Lead Poaching Protection:** Real-time duplicate detection prevents "poaching" by blocking entry of existing contact details and attributing them to the original agent.
- **Strict RBAC:** Granular access control for Superadmins, Managers, Supervisors, Agents, and IT Admins.

### 📱 Telegram Bridge (Storage & Alerts)
- **Unlimited File Hosting:** Uses the **Telegram Bot API** as an "infinite" storage server for property photos and payment receipts, keeping database costs at zero.
- **Real-time Mobile Alerts:** Instant notifications for new hot leads and status changes pushed directly to the agent's Telegram.

### 💰 Financial & Workforce Operations
- **DP Tracker (Milestones):** Integrated ledger for tracking downpayment schedules and payment history.
- **Licensing Tracker:** Manages the onboarding funnel for new recruits from "Applicant" to "Licensed Professional."
- **Project Inventory:** Centralized gallery that serves as the data source for external landing pages.

---

## 🛠️ Tech Stack

- **Frontend:** Next.js 15 (App Router), TypeScript, Tailwind CSS, Shadcn/UI
- **Backend/DB:** Supabase (PostgreSQL, Row Level Security)
- **AI Engine:** Google Gemini
